# myRep
A CRM tool designed for Xfinity retail store employees to keep track of customer leads. 

Link Here: [myRep](https://hammed74.github.io/myRep/)

 ## **About myRep**
This web app enables Xfinity representatives to create and populate leads with customer interaction data. It facilitates swift entry of details such as name, phone number, email, reward status, primary objective and interest level immediately after customer interactions. Each lead generates a ranking based on entered data, allowing reps to sort leads by 'temp', visually represented on a color scale (e.g., bright red for hot leads and icy blue for cold ones). Sorting options include date of interaction and temp. The app also includes a search function to help reps easily locate customers.

## **Notables Features**

Temp - A measurement tool that calculates the 'heat' of a lead using a weighted algorithm. It processes lead information input to generate a visual representation of lead temperature.

Search Tool - allows users to filter through lead by a variety of categories such as name, reward level, phone number and product category.

Dark Mode - A switch that allows users to change between dark mode and light mode to reduce eye starin and/or increase readability.
